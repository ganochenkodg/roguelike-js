# JS Roguelike

## Intro.

Explore eternal dungeon, try to survive.


## Keyboard

| Key                   | Action                                |
|:----------------------|:--------------------------------------|
| Arrows, numpad arrows | Move your character.                  |
| a .. p                | Choose item from inventory.           |
| 1 .. 9                | Use skill/magic.                      |
| \                     | Pick up item.                         |
| >                     | Move on the next level of dungeon     |
| <                     | Move on the previous level of dungeon |
| ?                     | Print the help                        |
